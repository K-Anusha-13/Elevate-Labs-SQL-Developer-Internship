# Final Project – SQL Database Implementation

## Overview

This project represents the **final assignment of the Elevate Labs SQL Developer Internship**.
The objective of the project is to design, implement, and analyze a relational database using SQL while applying the concepts learned throughout the internship.

The project demonstrates practical usage of **database schema design, data manipulation, SQL queries, and database reporting**.

---

## Project Objectives

* Design a relational database structure
* Implement tables with proper relationships
* Insert and manage structured data
* Perform data retrieval using SQL queries
* Generate outputs to analyze the stored data
